Add the ability to add reference images to the 3D view.

The reference images should be added to the 3D view as a background image.
They should be positioned in the 3D view and can be scaled and rotated.
They should be able to be moved around the 3D view.
They should be able to be deleted.
They should be able to be saved and loaded.

Reference images should always be placed in a X, Y or Z plane.

User should be able to set the transparency of the reference image.

The user should be able to scale the image by selecting 2 points on the image and enter the real distance between them.


Make sure that the loading and calibration of reference images also can be done through the MCP. So we can have an agent do the task.

Make reference images selectable and movable in the 3D view.
